**openbao/configure.yml** (OPT-01, OPT-02):

1. Wrap the TLS cert generation + permissions tasks (lines 4-26) in a `block/rescue`:
   ```yaml
   - name: OpenBao | Configure | Generate and secure TLS certificate
     block:
       - name: OpenBao | Configure | Generate self-signed TLS certificate
         ansible.builtin.command: >
           openssl req -x509 -newkey rsa:4096
           -keyout {{ openbao_tls_key_file }}
           -out {{ openbao_tls_cert_file }}
           -days 365 -nodes
           -subj "{{ openbao_tls_self_signed_subject }}"
         args:
           creates: "{{ openbao_tls_cert_file }}"
         when: openbao_tls_self_signed
         tags: [openbao, tls]

       - name: OpenBao | Configure | Set TLS certificate permissions
         ansible.builtin.file:
           path: "{{ item }}"
           owner: "{{ openbao_user }}"
           group: "{{ openbao_group }}"
           mode: "0600"
         loop:
           - "{{ openbao_tls_cert_file }}"
           - "{{ openbao_tls_key_file }}"
         tags: [openbao, tls]
     rescue:
       - name: OpenBao | Configure | TLS certificate generation failed - clean up partial files
         ansible.builtin.file:
           path: "{{ item }}"
           state: absent
         loop:
           - "{{ openbao_tls_cert_file }}"
           - "{{ openbao_tls_key_file }}"
         tags: [openbao, tls]

       - name: OpenBao | Configure | Fail with actionable message
         ansible.builtin.fail:
           msg: >-
             TLS certificate generation failed. Partial files have been cleaned up.
             Check that openssl is installed and {{ openbao_tls_self_signed_subject }} is valid.
             Re-run the playbook to retry.
         tags: [openbao, tls]
     when: openbao_tls_enabled
     tags: [openbao, tls]
   ```
2. Add `module_defaults` to a separate block wrapping the remaining deploy tasks (Deploy OpenBao configuration, Deploy backup script):

   ```yaml
           group: root
           mode: "0750"
         when: openbao_backup_enabled
         tags: [openbao, backup]
   ```
