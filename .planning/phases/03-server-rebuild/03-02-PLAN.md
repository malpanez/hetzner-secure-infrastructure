**INFRA-04 -- Both WordPress sites accessible:**
```bash
curl -sI https://twomindstrading.com | head -5
curl -sI https://academy.twomindstrading.com | head -5
```
Expected: HTTP/2 200 or 301 with `server: cloudflare` header.

**OPS-01 -- Valkey maxmemory 512MB:**
```bash
ssh malpanez@SERVER_IP "sudo valkey-cli CONFIG GET maxmemory"
```
Expected: Second line shows `536870912` (512MB in bytes).

**OPS-02 -- MariaDB binary logging active:**
```bash
ssh malpanez@SERVER_IP "sudo mysql -e \"SHOW VARIABLES LIKE 'log_bin';\""
ssh malpanez@SERVER_IP "sudo mysql -e \"SHOW MASTER STATUS;\""
```
Expected: `log_bin | ON` and a row with File and Position values.

**OPS-04 -- Fail2ban jails watching both nginx paths:**
```bash
ssh malpanez@SERVER_IP "sudo fail2ban-client status"
ssh malpanez@SERVER_IP "sudo fail2ban-client status nginx-main"
ssh malpanez@SERVER_IP "sudo fail2ban-client status nginx-academy"
```
Expected: Jail list includes `sshd`, `nginx-main`, `nginx-academy`. Both nginx jails return stats.

**OPS-05 -- AppArmor complain mode active:**
```bash
ssh malpanez@SERVER_IP "sudo aa-status"
```
Expected: Profiles loaded in complain mode (sshd, fail2ban-server). No profiles in enforce mode.

**OpenBao primary operational (INFRA-03 continued):**
```bash
ssh malpanez@SERVER_IP "export VAULT_ADDR=https://127.0.0.1:8200 VAULT_SKIP_VERIFY=1; bao status -format=json" | python3 -c "import sys,json; d=json.load(sys.stdin); print('UNSEALED' if not d['sealed'] else 'SEALED')"
```
Expected: `UNSEALED`

**MariaDB databases and users (INFRA-04 detail):**
```bash
ssh malpanez@SERVER_IP "sudo mysql -e \"SHOW DATABASES;\" | grep wordpress"
ssh malpanez@SERVER_IP "sudo mysql -e \"SELECT user,host FROM mysql.user WHERE user LIKE 'wp_%';\""
```
Expected: `wordpress_main`, `wordpress_academy`, `wp_main`, `wp_academy` all present.
